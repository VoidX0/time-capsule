## 添加摄像头
- 📹 在`摄像头`页面点击`➕ 新增摄像头`按钮
- 📝 填写摄像头信息(名称、[目录](#视频存储位置)、[解析模板](#视频名称解析模板))
- 💾 点击`保存`按钮完成添加

### 视频存储位置
- 📂 视频存储位置是基于[CameraPath](../../deploy/config/#systemoptions配置)的相对路径
- 例如：配置项中基础路径默认为`/camera`, 则在此处输入`test`表示视频实际目录为`/camera/test`

### 视频名称解析模板
- 🧩 此模板用于解析摄像头视频文件名中的时间信息
- ❗ 如果使用该模板解析失败，将尝试使用`ffmpeg`获取视频的创建时间
- 🔑 支持变量: `{start}`(视频开始时间), `{end}`(视频结束时间)
- ✨ 支持通配符: `*`(匹配任意字符)

#### 常用模板示例
- **默认模板** 文件名: `00_20250804181104_20250804182349.mp4` 模板: `*_{start:yyyyMMddHHmmss}_{end:yyyyMMddHHmmss}`
- 文件名: `camera5_start20250804_end20250805_extraInfo` 模板: `*_start{start:yyyyMMdd}_end{end:yyyyMMdd}_*`
- 文件名: `video_2025080412_2025080413.mp4` 模板: `video_{start:yyyyMMddHH}_{end:yyyyMMddHH}`

## 同步元数据索引

### 自动同步
- 🤖 每次同步元数据时，会同时同步所有视频片段的缩略图缓存
- ⏲️ 已添加的摄像头将根据[CronSyncAndCache](../../deploy/config/#systemoptions配置)配置的时间自动同步元数据索引

### 手动同步
- 🛠️ 在`摄像头`页面点击对应摄像头的`🔄️`，手动触发同步任务

## 修改或删除摄像头
- ✏️ 在`摄像头`页面点击对应摄像头的`✏️`按钮修改摄像头
- 🗑️ 点击`🗑️`按钮删除摄像头
- ⚠️ 注意：删除摄像头不会删除原始视频文件，只会删除该摄像头的相关配置、元数据与缓存
